Start_EIBP.py - will upload EIBP_code from the local to the remote location, and the runs the protocol on all nodes in the topology. 

EIBP_stop_process.py – will stop the EIBP that was running on all nodes in the topology.

Run_EIBP.py – will run EIBP on all nodes in the topology. 

EIBP_stop.py – will stop the EIBP that was running on all nodes in the topology and removes the EIBP code from the remote location. 

Clear.py – will erase all the contents in the eibllog file.

Credentials&Path.txt – contains all the credentials and paths required to run the protocol. 

UnreservedRSPEC.xml – rspec file from the slice.
